# Assignment 2

This folder contains the code and report for the analysis in Assignment 2.

## Files

- `analysis_report.ipynb`  
  The main report notebook. It contains the analysis, figures, and written interpretation of the results.

- `analysis_functions.py`  
  Helper functions used by the notebook. This file handles model setup, prediction, CAM generation, layer activations, and visualization.


## Requirements

To run the files, you need:

- Python 3.12
- `torch`
- `torchvision`
- `torchcam`
- `pillow`
- `matplotlib`


The notebook imports functions from `analysis_functions.py`, so both files need to remain in the same folder.
